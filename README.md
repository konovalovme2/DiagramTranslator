# DiagramTranlator
Simple translator of block-diagrams to text description by using some ML methods (CV. OCR) for ITMO Megaschool. Made by Timofey Khramov and Konovalov Mikhail (team "гАИшники")

To start run command:

pip install -r requirements.txt

Then, in two different terminals:

1. streamlit run app.py
2. python ./server/main.py
